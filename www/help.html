Current jobs

The Current section lists all backup jobs that are scheduled or active. Each entry shows the job name, the path being backed up, and the scheduled date and time. If no jobs have been scheduled yet, the list will be empty.

Schedule a backup

Click Schedule to create a new backup job. Fill in the following fields:

  • Name — a short label for the backup job (for example, “Home directory” or “Project files”).
  • Path — the full path to the file or directory you want to back up (for example, /home/user/documents).
  • Date — the month, day, and year on which the backup should run.
  • Time — the hour and minute at which the backup should start.

Click the submit button to add the job. The new job will then appear in the Current list.

Restore from a previous backup

Click Restore to see a table of completed backup jobs. Each row shows the job ID, name, path, date, time, and the backup archive file. Select the job you want to restore and submit the form. FlashBack will unpack the archive and restore the files to their original location.

Settings

Application settings are configured in the FlashBack configuration file (typically /etc/flashback/flashback.conf). Restart the FlashBack daemon after making changes for them to take effect.
